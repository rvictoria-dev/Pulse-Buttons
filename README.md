# Pulse-Buttons

### ✨ About

Pulse Buttons is a collection of 6 animated buttons built with pure CSS. Each one uses `@keyframes` to create a different pulse effect — scale, ring, double ring, opacity, border, and color. No JavaScript, just HTML and CSS.
